Definitions

a - Found coverage is defined as the number of structures found / number of structures enumerated

b - Spray coverage is defined as number of structures sprayed / number of structures found

c - Start time is determined from the time the first form is completed per day per operator

d - Finish time is determined from the time the last form is started per day per operator
